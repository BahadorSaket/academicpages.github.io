The goal of this study is to measure User Interface (UI) similarities from the user's perspective.

In this study, you will be asked to complete 10 tasks. For each task, single or multiple sketches of an app are shown on top with some additional apps below that. Your task is to compare the top app with the below ones and rank them based on the similarity (from the most similar to the least similar). Each app is labeled with a number (1,2,3,..) and each screen of an app is labeled with an alphabet (A,B,C,..). When doing comparison, note that you should compare the UIs with the same alphabets against each other if it exists. For instance, in the following example, for each app you should compare screen A/B of the sketch on the top with screen A/B of the below apps. Then, you have to decide which apps are more similar to the top one in overall and provide the ranking based on that. You just need to put the ranking in the provided text box for each task. For this example, the ranking can be 3 1 2 or 1 2 3 or 2 1 3, etc.

By UI similarity, we mean similarity of:
1) Number of widgets (e.g., number of buttons/sliders)
2) Type of widgets (e.g., button vs. slider)
3) Text associated with widgets
4) Location of widgets (e.g., top-left of the UI vs. center of the UI )
5) Size of widgets (e.g., small button vs. large button)

Please note, colors and images contained in the interface should not be the criteria during your evaluation.
